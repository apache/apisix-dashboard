## 监控

> Metrics 页面是以 iframe 方式打开 Grafana， Gragana 以 iframe 打开需要配置 [`allow_embedding=true`](https://grafana.com/docs/grafana/latest/administration/configuration/#allow_embedding)支持,该值默认为`false`  否则浏览器将无法正常渲染 Grafana 页面
