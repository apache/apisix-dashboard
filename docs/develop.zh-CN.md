# Dashboard 开发

## 前端开发

1. 确保你的设备已经安装了 `Node.js(version version 10.0.0+)`。

2. 安装 [yarn](https://yarnpkg.com/)。

3. 安装依赖:

```sh
$ yarn install
```

4. 若需要修改 manager-api 地址，请访问 `config/proxy.ts` 文件。

5. 启动开发模式

```sh
$ yarn start
```
